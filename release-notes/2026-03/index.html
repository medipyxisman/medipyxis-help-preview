March 2026 release notes

release-note Clinician Intake Admin Billing Business development Vendor Updated 2026-04-24

Changes shipped between 2026-03-01 and 2026-03-31.

This release contained no user-facing feature changes. Work shipped in March was limited to infrastructure maintenance, dependency updates, and back-end data migrations. No module documentation was affected.

Several internal engineering tasks were resolved with resolution dates in late March (MED-1596, MED-1676, MED-1432, MED-1215, MED-1595, MED-1124, and others); those items did not produce observable UI or workflow changes for end users and are therefore not listed individually. If you need the technical details for an audit or integration review, contact your implementation manager with the ticket key.

User-facing changes from the next deploy cycle are documented in April 2026 release notes.
