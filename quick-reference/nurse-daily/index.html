Nurse daily cheat sheet

reference ⏱ 2 min read Clinician Updated 2026-04-24

One page. Print it or keep it pinned. Everything you do in a typical day.

Start of day

Task Where Shortcut
See today's stops Patient Management → today's list —
Preview the week Fleet Calendar → week view —
Check action items Dashboard → right panel —
Clock in to duty Header → On Duty toggle —

During a visit

Task Where Shortcut
Start visit Patient card → Start Visit —
Jump between wizard steps Left wizard rail ← / →
Add wound photo Wound Assessment step → camera icon —
Scan graft UIN Procedures & Supplies → Scan —
Save draft Wizard auto-saves every 15s; manual: top-right ⌘S / Ctrl+S
Sign & complete Final step → Sign & Complete —

Offline

  • Yellow Offline banner = you're fine. Keep charting.
  • Photos save locally, upload when online.
  • Syncing status lives in the header next to your name.
  • Never force-quit during a sync — it's fast but it's a real sync.

Shortcuts that save real time

Shortcut Action
⌘K / Ctrl+K Quick Jump — patient, visit, page
⌘S / Ctrl+S Save draft
⌘↵ / Ctrl+Enter Submit current wizard step
⌘/ / Ctrl+/ Help for current page

Who to ping when

Problem Channel
App bug Team Chat → #support
Scheduling conflict Your ops manager via Team Chat DM
Clinical question Team Chat → #clinical
Credential about to expire Already alerted — Dashboard → My Credentials

End of day

  1. All visits signed? Check Action Items is empty.
  2. Log mileage: Dashboard → Mileage & Expenses.
  3. Flip On Duty to Off Duty.
